About the Website
Frontend (User Interface)
These are used to design and control what users see:
HTML – Creates the structure of the website
CSS – Styling, layout, responsiveness, animations
JavaScript – User interactions, form handling, API calls
VS Code – Code editor for writing HTML, CSS & JS
Browser (Chrome/Edge) – Testing and debugging
Backend (Logic & Processing)
Used to handle requests and connect with AI services:
JavaScript (Node.js – optional) – Backend logic (if used)
Fetch API / Axios – Sends requests to AI image API
Environment Variables (.env) – Secure API key storage
(If backend is not used, API is called directly from JavaScript)
AI Image Generation API
Used to generate images from text:
OpenAI Image API / Stable Diffusion API / Other AI Image APIs
Converts text prompt → AI-generated image
Requires API Key for authentication
Hosting & Deployment
Used to make the website live:
Git & GitHub – Version control and code storage
